# Sreeyuth Chowdary — Portfolio

Personal portfolio website showcasing my work as a Software Developer and Computer Science student at Northwest Missouri State University.

## Live Site

[View Portfolio](https://sreeyuthgunnam.github.io/Portfolio/)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Flexbox, Grid, animations
- **JavaScript** — Vanilla JS for interactions
- **Google Fonts** — Inter, Space Grotesk

## Structure

```
Portfolio/
├── index.html          # Main page
├── css/styles.css      # All styling
├── js/script.js        # Interactions & animations
├── assets/             # Images & icons
│   ├── profile.jpg
│   ├── paw.png
│   ├── N-Monogm-BW.png
│   ├── CCSC.png
│   └── [contact icons]
└── resume/Resume.pdf   # Downloadable resume
```

## Features

- Responsive design (mobile-first)
- Smooth scroll navigation
- Typewriter text animation
- Scroll reveal animations
- Hide/show navbar on scroll
- Frosted glass navbar effect
- Bearcat green theme (#00a86b)

## Deployment

This is a static site. Deploy to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting

## To Update Content

1. **Resume** — Replace `resume/Resume.pdf`
2. **Profile photo** — Replace `assets/profile.jpg`
3. **Projects** — Edit the projects section in `index.html`
4. **Contact** — Update email/phone in `index.html`
