# Sreeyuth Chowdary — Portfolio

Personal portfolio website showcasing my work as a Software Developer and Computer Science student at Northwest Missouri State University.

## Live Site

[View Portfolio](www.sreeyuth.com)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Flexbox, Grid, animations
- **JavaScript** — Vanilla JS for interactions
- **Google Fonts** — Inter, Space Grotesk

## Structure

```
Portfolio/
├── index.html          # Main page
├── css/styles.css      # All styling
├── js/script.js        # Interactions & animations
├── assets/             # Images & icons
```

## Features

- Responsive design (mobile-first)
- Smooth scroll navigation
- Typewriter text animation
- Scroll reveal animations
- Hide/show navbar on scroll
- Frosted glass navbar effect
- Bearcat green theme (#00a86b)
